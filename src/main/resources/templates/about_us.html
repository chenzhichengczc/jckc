少儿启蒙

     少儿启蒙，专业3-8岁少儿编程教育服务，形成启蒙意识，由浅入深，让小朋友具备一定的编程思想，促使大脑有自主发展能力，提高小朋友专注力，设计力，全方位发展。

启蒙理念

     金诚科创注重少儿互助启蒙形式，注重以讨论，一对一讲授，动手解决能力，传递前沿技术，提问互动等形式，让小朋友了解编程世界的美妙，以简单的小案例，小游戏，提高兴趣感，通过自己一步步能构建出“我的世界”。

启蒙小案例

    • 具备可玩性飞机大战 / 坦克大战 / 我的世界；
    • 模仿淘宝 / 京东 / 阿里巴巴页面搭建；
    • 二维码设计 / 二维码小知识 / 丰富二维码可玩度；
    • 经典算法题 / 脑力风暴 / PK大赛；
